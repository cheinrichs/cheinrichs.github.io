ATMZR

A link shortener API that I built to learn Golang. It uses Mux routing, and I built it with dependency injection so that all routes can be easily unit tested without accessing the database. The API is live at http://atmzr.herokuapp.com/

  • A link with a valid hash will be redirect to the original target link
  • CreateLink - creates a shortened link
  • linkStatistics - Allows a user to see how many times a link has been visited
